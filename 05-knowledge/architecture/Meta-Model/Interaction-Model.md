# Interaction Model

## Purpose
To establish the standard vocabulary (verbs) used to map the semantic relationships between objects. This standardizes diagrams, traceability matrices, and AI reasoning.

## Standard Verbs

* **Creates / Produces:** Originates a new object as a direct output.
* **Consumes / Requires:** Uses an object as a strict input or dependency.
* **Approves:** Grants a state transition through formal authority.
* **Owns:** Holds ultimate accountability for the object's lifecycle.
* **Depends On:** Relies on another object for functionality, execution, or transition.
* **Verifies:** Proves technical correctness or adherence to specifications.
* **Validates:** Proves business alignment and capability intent.
* **Replaces / Supersedes:** Deprecates a previous version or competing object.
* **Archives:** Moves an object to a terminal, read-only state for historical preservation.
