# Naming Convention Standard



## Document Naming Rule

> **File names should describe their content. Document identifiers should uniquely identify the document. These two responsibilities must remain separate.**

Examples:
- Versioning.md -> ID: EOS-003
- Capability-First.md -> ID: ADR-001
